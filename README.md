# 📊 Sales Prediction Using Regression Analysis

This project explores the relationship between **product sales** and key economic factors including **price, advertising spend, GDP per capita, and internet penetration** using both linear and log-log regression models.

## ✨ What Makes This Special

- **Simple, Clean Code** – Easy-to-follow implementation in Python/R
- **Two Models Compared** – Linear vs. Log-Log (spoiler: log-log wins! 🏆)
- **Beautiful Visualizations** – Diagnostic plots that tell the story
- **Real Economic Insights** – Price elasticity, advertising ROI, and more

## 📈 Key Findings

- Log-log model explains **95.9%** of sales variation (vs. 71.1% for linear)
- **1% price increase** → **1.66% sales decrease** (elastic demand)
- **Advertising works!** 1% more spend → 0.61% more sales
- Wealthier markets = more sales (income elasticity of 1.08)

## Built With

- Python (pandas, statsmodels, matplotlib/seaborn, Excel)
- Jupyter Notebook / RMarkdown
